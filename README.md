# ML Project - NYC Yellow Taxi Data Analysis

Analysis of NYC Yellow Taxi Trip Records (November 2025) covering data exploration,
preprocessing, statistical analysis, classification, regression, clustering, NLP and more.

## Dataset

Download `yellow_tripdata_2025-11.parquet` from [NYC TLC](https://www.nyc.gov/site/tlc/about/tlc-trip-record-data.page)
and place it in the project root.

## Requirements

```
pip install pandas numpy matplotlib seaborn scikit-learn nltk wordcloud pyarrow scipy
```

## Usage

Open `ML_Lab_Report.ipynb` in Jupyter Notebook or VS Code and run all cells.
